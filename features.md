# FlowCredit Features Documentation

---

## 🔹 Core Features

### 1. Supplier Onboarding

* User registration and login
* Profile creation
* Business details capture

---

### 2. Transaction Data Input

* Manual entry of M-Pesa transactions
* Upload of transaction statements (optional)
* Data validation and formatting

---

### 3. Credit Scoring System

* Analysis of:

  * Transaction frequency
  * Cash flow consistency
  * Incoming vs outgoing payments
* Generates a simple credit score

---

### 4. Loan Offer System

* Displays eligible loan amount
* Shows repayment terms
* Allows supplier to accept or decline

---

### 5. Loan Disbursement

* Funds sent via M-Pesa (or simulated in MVP)
* Instant or near-instant processing

---

### 6. Automated Repayment

* Deduction from incoming payments
* Tracking of repayment progress
* Status updates (paid / pending)

---

### 7. Factory Dashboard

* View supplier activity
* Monitor supplier reliability
* Track financed suppliers

---

### 8. Risk Management System

* Lending limits based on cash flow
* Supplier scoring thresholds
* Payment tracking safeguards

---

## 🔸 MVP Features (Hackathon Scope)

* Basic authentication system
* Manual transaction input
* Simple scoring logic (rule-based)
* Loan simulation
* Mock M-Pesa integration
* Basic dashboards

---

## 🔸 Advanced Features (Future Scope)

* Real-time M-Pesa API integration
* AI-powered credit scoring
* Invoice / Purchase Order financing
* Automated payment splitting
* Notifications (SMS / Push)
* Multi-currency support

---

## 🎯 Feature Goals

Each feature is designed to:

* Improve financial access
* Reduce lending risk
* Simplify supplier experience
* Strengthen supply chain reliability

---

## 📌 Summary

FlowCredit combines **fintech + supply chain management** to create a seamless financing experience powered by real transaction data.
